# Delhi–Dehradun Expressway 

## 📌 Overview
This project simulates a smart highway system inspired by the Delhi–Dehradun Expressway.  
It demonstrates how **advanced Data Structures and Algorithms (DSA)** can solve real-world problems such as route optimization, toll management, and traffic analysis.

---

## 🎯 Objectives
- Implement real-world applications of DSA  
- Simulate a highway navigation system  
- Optimize routes based on time, distance, and toll  
- Manage toll data efficiently  
- Perform fast queries using advanced data structures  

---

## 🧩 Concepts Used
- **Graph**: Cities → Nodes, Roads → Edges, Weights → Distance, Time, Toll  
- **Dijkstra Algorithm**: Shortest path (time, distance, toll)  
- **Bellman-Ford Algorithm**: Detects negative cycles (fuel arbitrage)  
- **Kruskal + DSU**: Minimum Spanning Tree  
- **Fenwick Tree**: Efficient range queries  
- **Bloom Filter**: Vehicle toll pass check  
- **Heap / Priority Queue**: Dijkstra & Top-K queries  
- **Hashing**: Fast lookups  

---

## 🚀 Features
- Shortest Path Finder (Path, Time, Distance, Toll)  
- Minimum Spanning Tree (Optimized road network)  
- Fuel Arbitrage Detection  
- Toll Management System  
- Top-K Toll Plazas  
- Range Query System  

---

## 🛠️ Tech Stack
- **Language:** C++  
- **Libraries:** STL, File Handling  
- **Algorithms:** Graph-based (Dijkstra, Bellman-Ford, Kruskal, DSU, Fenwick Tree, Bloom Filter)  

---

## 📂 System Architecture
- **Input:** `expressway.txt` (cities, roads, weights)  
- **Processing:** Graph creation + algorithm execution  
- **Output:** CLI-based results
